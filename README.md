# Tetris

It is a variation on Tetris, ported to [JavaScript](https://en.wikipedia.org/wiki/JavaScript).

## Play Online

Visit [tetris.nylira.com](http://tetris.nylira.com) to play online.

## Play Locally

    npm install
    node server.js
